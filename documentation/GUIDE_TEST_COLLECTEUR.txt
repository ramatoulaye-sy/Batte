GUIDE RAPIDE - TESTER L'INSCRIPTION COLLECTEUR
===============================================

ETAPE 1: EXECUTER LE SQL (5 min)
---------------------------------
1. Va sur Supabase → SQL Editor
2. Copie/colle le contenu de: database/create_user_profiles_table.sql
3. Clique "Run"
4. Verifie "Success" ✅

ETAPE 2: REDEMARRER L'APP (2 min)
----------------------------------
Dans le terminal:
1. Ctrl+C (arreter l'app)
2. flutter run (relancer)
3. Attends que ca compile

ETAPE 3: TESTER (3 min)
------------------------
1. L'app demarre
2. Login avec ton compte existant
3. Tu DOIS voir l'ecran: "Bienvenue sur Batte !"
4. Deux choix:
   - "Utilisateur" (pour recycler)
   - "Collecteur" (pour collecter)

ETAPE 4: CREER PROFIL COLLECTEUR
---------------------------------
1. Clique sur "Collecteur"
2. Remplis le formulaire:
   - Business: "Mon Recyclage"
   - Licence: "RC-GN-2024-999"
   - Vehicule: "Camion Toyota"
   - Rayon: 10
3. Clique "Creer mon profil collecteur"
4. Tu verras le DASHBOARD COLLECTEUR avec:
   - Tes stats
   - Toggle Disponible/Indisponible
   - Gains totaux
   - Etc.

SI CA NE MARCHE PAS:
--------------------
1. Verifie que le SQL est execute (Etape 1)
2. Verifie les logs du terminal
3. Redemarre l'app

POUR REVENIR EN UTILISATEUR:
-----------------------------
Une fois que tu as les 2 profils:
- Au prochain login, tu verras un ecran de CHOIX
- Tu pourras choisir: Utilisateur OU Collecteur
- Un meme compte peut avoir les 2 profils !

C'EST TOUT ! ✅
